## Example: Markdown Snippet

You can easily add reusable markdown snippets to your documentation. Here’s a simple example of how to define and use a snippet:

```mdx
// snippets/warning.mdx
> **Warning:** Make sure to save your work frequently!
```

To use this snippet in your docs, simply include:

```mdx
import Warning from '../snippets/warning.mdx'

<Warning />
```

This will render the warning message anywhere you need it, helping you keep your docs consistent and maintainable.
